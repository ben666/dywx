组织介绍 组织架构 党员须知

梦想小镇党委依托孵化器、行业、社团为主体，以更加灵活的方式设立党组织，形成较为稳定的组织格局。建立了一套“四联”党群工作互融机制，以小镇党委和各党支部为核心，以公会、妇联、团委、商会等组织为依托，全面推行“工作联动、队伍联盟、组建联手、资源联动”的“四联”运作方式，实现党群工作一体化。目前，梦想小镇党委下设13个党支部，其中联建式支部1个，苗圃式支部6个，单建式支部4个，联动式支部2个。

question 1. 党的最大政治优势是什么？

答：密切联系群众。

question 2. 党最大危险是什么？

答：脱离群众。

question 3. “三会一课”具体指什么？

答：支部委员会：一般每月召开一次；支部党员大会：一般每季度召开一次；党小组会：一般每月召开一至二次；党课：一般每季度举行一次。

question 4. 党的纪律处分有哪些？

答：警告、严重警告、（ 撤销党内职务） 、 （留党察看） 和开除党籍。

question 5. “三讲教育”指的是哪三讲？

答：讲学习、 讲政治 、 讲正气。

question 6.什么情况下使用党员组织关系介绍信？

答：党员外出地点或工作单位相对固定，外出时间6个月以上的，一般应当开具中国共产党党员组织关系介绍信。
